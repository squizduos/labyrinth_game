# Лабиринт сокровищ

Текстовая adventure игра на чистом Python. Исследуйте лабиринт, собирайте предметы, решайте загадки и найдите сокровище!

## Установка

```bash
make install
```

или

```bash
poetry install
```

## Запуск

```bash
make project
```

или

```bash
poetry run project
```
